# Day 2 - C# Fundamentals I: Types, Variables & Control Flow

## Topics Covered

- Value Types vs Reference Types
- Variables and Type Inference (var)
- Naming Conventions
- Control Flow (if, switch, switch expression, loops)
- Nullable Reference Types

## Hands-On Lab

### Task 1
Created a console application with:
- 3 Value Types
- 3 Reference Types
- Printed each type using GetType().

### Task 2
Demonstrated the difference between Value Types and Reference Types by copying variables and showing the behavior before and after modification.

### Task 3
Implemented a grade classifier using a Switch Expression with multiple score ranges.

### Task 4
Created a program that safely handles nullable user input using Nullable Reference Types.
